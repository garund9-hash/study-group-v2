## Implementation Workflow
- **Plan First:** Always propose a implementation plan before writing code.
- **Approval & Documentation:** Once I approve a plan, you must automatically create or update a `PLAN.md` file in the root directory.
- **Plan Content:** - Summary of changes
  - Implementation checklist with [ ] task markers
  - Potential side effects
- **Update Cycle:** Mark tasks as [x] as you complete them during the coding phase.
